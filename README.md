# Python JSON Data Analysis

This project analyzes a social network dataset stored in JSON format using Python.

## Features
- Data loading from JSON files
- Data cleaning and preprocessing
- "People You May Know" friend recommendation system
- Page recommendation system based on shared interests

## Technologies Used
- Python
- JSON
- Jupyter Notebook

## Project Files
- 01introduction.ipynb – Introduction to the dataset
- 02data_cleaning.ipynb – Data cleaning and preprocessing
- people_you_may_know.ipynb – Friend recommendation algorithm
- pages_you_might_like.ipynb – Page recommendation system
- data.json – Original dataset
- cleaned_data2.json – Cleaned dataset

## Purpose
This project demonstrates how Python can be used to analyze structured data and implement basic recommendation systems similar to social media platforms.
